================================

Solution: Kill any existing Node processes, then run ./start.sh

Problem: Can't see surveys
Solution: Refresh browser (Ctrl+F5), check backend console

Problem: Forms not rendering
Solution: Check browser console for errors, verify schema is valid JSON

Problem: Responses not showing
Solution: Fill all required fields (marked with *), check console

Problem: Styles look wrong
Solution: Clear browser cache, rebuild: npm run build

Problem: API not responding
Solution: Ensure backend is running: cd backend && npm start
